Project Summary

Overview

Whether you are looking to create a garment from scratch or just sew on a missing button, sewing remains a very useful skill to have. Learning how to sew, however, can be an intimidating and frustrating experience—whether you’re a complete newbie learning to stitch or an expert teaching yourself a new technique. To make learning how to sew more accessible, we created an app to help guide sewers through the process. Stitchio also provides resources to aid in creating a more holistic engagement with the craft.

User Testing

After creating a quick prototype we tested our concept with two users to validate our navigation tab, test the onboarding flow and find any missed edge case scenarios. We got the following feedback -‍

  • The beginner, intermediate and advanced sewing levels need to be more clearly defined during the onboarding
  • Library tab in navigation is too vague; leaving users unsure of what they would find there
  • For machine set up there are too many different types of machine. It would be easier to focus on set up in terms of settings for different fabric types.
  • Add a category for notions which would include tools in the libraries page.
  • Need for differentiation between: doing a project and learning a technique - 2 different tasks
